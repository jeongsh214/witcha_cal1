캐릭터 스탯 비교기

캐릭터 스탯을 그대로 붙여넣으면, 등급을 기준으로 CSV에서 기준 스탯을 불러와 비교한다.
숫자는 고정값으로 처리하고, ~가 들어가면 범위값으로 처리한다.

캐릭터 목록

characters.csv를 불러와 목록을 관리한다. 상태 변경과 삭제는 현재 페이지 목록에 반영되며, 다운로드한 CSV를 GitHub에 덮어쓰면 공유된다.

캐릭터 입력

입력 규칙
- 줄마다 항목명 + 값 형식이면 읽는다.
- 탭, 콜론(:), 여러 칸 공백을 구분자로 처리한다.
- 숫자만 있으면 고정값으로 처리한다.
- 예: 4~7 은 최소 4, 최대 7로 처리한다.
- 범위값은 최소/최대를 각각 따로 비교해 색도 따로 표시한다.
- 기준값은 같은 폴더의 CSV 파일에서 자동으로 불러온다.
- 캐릭터 목록 저장용 CSV 구조는 이름,등급,체력,공격력,방어력,치명타 확률,명중률,민첩,기각여부 이다.
